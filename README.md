# 🎲 Dicee Game

A simple interactive dice game built using HTML, CSS, and JavaScript.

Whenever the page is refreshed, both players roll a random dice and the winner is displayed dynamically using JavaScript DOM manipulation.

## 🚀 Features
* Random dice generation using JavaScript
* Dynamic dice image updates
* Winner announcement logic
* DOM manipulation with JavaScript
* Responsive and clean UI

## 🛠️ Technologies Used
* HTML5
* CSS3
* JavaScript (DOM)

## 🎯 How It Works
1. Player 1 gets a random number between 1 and 6.
2. Player 2 gets a random number between 1 and 6.
3. Dice images change dynamically based on generated numbers.
4. The winner is displayed at the top:

   * Player 1 Won
   * Player 2 Won
   * Draw

## ▶️ How to Run
1. Clone the repository:

```bash
git clone https://github.com/Sanathanimv/Dicee-Game.git
```

2. Open the project folder.

3. Run `dicee.html` in your browser.
